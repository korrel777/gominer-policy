Privacy Policy

App: Go Miner Tools ( for gomining miner wars )

Effective date: February 16, 2026

1. What This Extension Does

This Chrome extension provides live analytics, side-panel tools, and alerts for GoMining Miner Wars pages.

2. Data We Process

To provide features, the extension may process:

  • Game-related page data from gomining.com and related subdomains.
  • Live game events (including API/WebSocket-derived events) needed for real-time features.
  • Local settings and cached data stored in Chrome extension storage.
  • License/status data through Supabase endpoints used by the app.

3. How Data Is Used

  • Render live dashboard panels and statistics.
  • Support notifications and alerting features configured by the user.
  • Store user preferences and session state for extension functionality.
  • Validate license/subscription access where applicable.

4. Data Sharing

We do not sell personal data. Data is only sent to services required for extension functionality (for example, configured GoMining/Supabase/notification endpoints).

5. Data Retention

Locally stored extension data remains on your device until you clear extension data or remove the extension. Server-side retention depends on the configured backend services.

6. Security

We use reasonable technical measures to protect data, but no system can be guaranteed 100% secure.

7. Your Choices

  • You can uninstall the extension at any time.
  • You can clear extension storage through Chrome extension settings.

This policy may be updated from time to time. The latest version will always be published at this URL.
